Install Choco - using Powershell
Install mkcert and openssl as Administrator
- choco install mkcert
- choco install openssl -y

Generate android  CA

- mkcert -CAROOT

Generate cert for android and iphone from mkcert folder

- openssl x509 -outform der -in rootCA.pem -out rootCA.cer
